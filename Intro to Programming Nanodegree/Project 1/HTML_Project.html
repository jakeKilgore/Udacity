The Internet

The internet is a web‑like connection of computers, split between servers and clients. In general, servers host files while clients use internet browsers to access them. Internet browsers use a protocol called HTTP to request files from servers and then use HTML to interpret these files into a human‑readable format as a web page.

HTML

HTML stands for Hyper Text Markup Language. It is a type of document containing:

  • Text
  • Markup
  • References
  • Links

Text

Text is simply the information conveyed with written language.

Markup

Markup is the grammar system used by internet browsers to group information together and display it to the user. In HTML, markup is written as elements which consist of a opening tag, contents, and closing tag.

<TAG>contents</TAG>
Some tags, referred to a void tags, have no content and no closing tag.
<TAG>
Tags may also have attributes which define information about the tag.
<TAG ATTR="value">contents</TAG>

References

HTML can make a reference to another file on the server or on the internet such as a picture and the client's internet browser will display it on the webpage.

<img src="url" alt="alttext">

Links

HTML can link webpages together by placing clickable text on the web page which will direct the client to another web page on the internet.

<a href="url">Clickable Text</a>
This is a link to Udacity's homepage.

Inline vs. Block

Some HTML elements are classified as being inline while others are classified as block. The main difference is that inline elements, such as span will not affect its content's position on the webpage while block elements such as div will place the element's content on a new line and create an invisible box around the content where other content cannot be placed.

An Example

HTML, like all computer science languages, is incredibly precise, avoiding the inevitable ambiguities of natural languages. It is also intended to be readable by humans and thus has conventions, such as indentation, that must be followed.

The following is a basic example of valid HTML code for a web page:

<!‑‑ This is a comment and will be ignored by the browser ‑‑>

<!DOCTYPE html>
<html>
<head>
 <title>Title of Web Page</title>
</head>
<body>
 <h1>Heading</h1>
 <p>
  Paragraph of text.
 </p>
 <img src="Cal.jpg" alt="My Dog">
 <br>
 <a href="https://www.w3schools.com/">
  A useful resource
 </a>
</body>
</html>
