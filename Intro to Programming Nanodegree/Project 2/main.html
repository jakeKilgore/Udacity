The Internet

The internet is a web‑like connection of computers, split between servers and clients. In general, servers host files while clients use internet browsers to access them. Internet broswers use a protocol called HTTP to request files from servers and then use HTML to interpret these files into a human‑readable format called a web page.

HTML

HTML stands for Hyper Text Markup Language. It is a type of document containing:

  • Text
  • Markup
  • References
  • Links

Text

Text is simply the information conveyed with written language.

Markup

Markup is the grammar system used by internet browsers to group information together and display it to the user. In HTML, markup is written as elements which consist of an opening tag, contents, and closing tag.

Tags

 <TAG> contents </TAG>
Some tags, referred to a void tags, have no content and no closing tag.

Void Tags

 <TAG>
Tags may also have attributes which define information about the tag.

Attributes

 <TAG  ATTRIBUTE= "value"> contents </TAG>

References

HTML can make a reference to another file on the server or on the internet such as a picture and the client's internet browser will display it on the webpage.

File Reference

 <TAG  src= "url">

Example Image

 <img  src= "Cal.jpg"  alt= "My dog">

Links

HTML can link webpages together by placing clickable text on the web page which will direct the client to another web page on the internet.

Links

 <a  href= "url"> Clickable Text </a>

Example Link

 <a  href= "https://www.udacity.com"  target= "_blank">
This is a link to Udacity's homepage.

Inline vs. Block

Some HTML elements are classified as being inline while others are classified as block. The main difference is that inline elements, such as span will not affect its content's position on the webpage while block elements such as div will place the element's content on a new line and create an invisible box around the content where other content cannot be placed.
